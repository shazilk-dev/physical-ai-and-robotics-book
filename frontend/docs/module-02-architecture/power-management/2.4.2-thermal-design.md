## Heat Generation in Humanoid Robots

### Power Dissipation = Heat

**Fundamental Law:**
```
Heat Generated (W) = Power Consumed - Useful Work

For electrical components (motors, CPUs, etc.):
Efficiency Ã— Input Power = Useful Work
(1 - Efficiency) Ã— Input Power = Heat
```

### Fan Control (PWM)

**Why Control Fan Speed?**
- Save power when cooling isn't needed
- Reduce noise during idle
- Extend fan lifespan

**PWM Control Circuit:**
```python
import RPi.GPIO as GPIO

# Setup
GPIO.setmode(GPIO.BCM)
fan_pin = 18  # GPIO 18 supports hardware PWM
GPIO.setup(fan_pin, GPIO.OUT)

# Create PWM object (25 kHz frequency)
fan_pwm = GPIO.PWM(fan_pin, 25000)
fan_pwm.start(0)  # Start at 0% duty cycle (fan off)

# Control fan speed based on temperature
def set_fan_speed(temp_celsius):
    """
    Adjust fan speed based on temperature
    < 50Â°C: Fan off
    50-70Â°C: Linear ramp from 0% to 100%
    > 70Â°C: 100% speed
    """
    if temp_celsius < 50:
        duty_cycle = 0
    elif temp_celsius > 70:
        duty_cycle = 100
    else:
        # Linear interpolation
        duty_cycle = ((temp_celsius - 50) / 20) * 100

    fan_pwm.ChangeDutyCycle(duty_cycle)
    return duty_cycle

# Example usage
import time
while True:
    # Read temperature (from Jetson thermal sensors)
    with open('/sys/devices/virtual/thermal/thermal_zone0/temp', 'r') as f:
        temp = int(f.read()) / 1000.0  # millicelsius to celsius

    speed = set_fan_speed(temp)
    print(f"Temp: {temp:.1f}Â°C, Fan: {speed:.0f}%")

    time.sleep(1)
```

**Result:** Fan runs at minimum speed most of the time, only ramps up under load (saves ~3-5W average power).

### Simplified Thermal Model (Python)

**Lumped Thermal Network:**

```python
import numpy as np
import matplotlib.pyplot as plt

class ThermalNode:
    """Single node in thermal network"""
    def __init__(self, name, heat_gen, thermal_mass, temp_init=25):
        self.name = name
        self.heat_gen = heat_gen  # W
        self.thermal_mass = thermal_mass  # J/K
        self.temp = temp_init  # Â°C
        self.connections = []  # (other_node, thermal_resistance)

    def add_connection(self, other_node, thermal_resistance):
        """Connect to another node via thermal resistance (K/W)"""
        self.connections.append((other_node, thermal_resistance))

    def update_temp(self, dt):
        """Update temperature based on heat flow (Euler integration)"""
        heat_in = self.heat_gen  # Internal heat generation

        # Heat flow from/to connected nodes
        for other_node, R_th in self.connections:
            heat_flow = (other_node.temp - self.temp) / R_th  # Q = Î”T / R
            heat_in += heat_flow

        # Temperature change: dT = (Q Ã— dt) / (m Ã— c)
        dT = (heat_in * dt) / self.thermal_mass
        self.temp += dT

# Example: Jetson Orin with heat sink
ambient = ThermalNode("Ambient", 0, 1e6, temp_init=35)  # Infinite thermal mass
jetson = ThermalNode("Jetson", 45, 100, temp_init=35)  # 45W heat, 100 J/K thermal mass
heatsink = ThermalNode("Heat Sink", 0, 500, temp_init=35)  # 500 J/K thermal mass

# Thermal resistances
R_jetson_heatsink = 0.5  # K/W (thermal paste + contact)
R_heatsink_ambient = 1.0  # K/W (natural convection)

jetson.add_connection(heatsink, R_jetson_heatsink)
heatsink.add_connection(ambient, R_heatsink_ambient)
heatsink.add_connection(jetson, R_jetson_heatsink)  # Bidirectional

# Simulation
time = []
jetson_temps = []
heatsink_temps = []

dt = 1.0  # 1 second time step
for t in range(0, 600):  # 10 minutes
    jetson.update_temp(dt)
    heatsink.update_temp(dt)

    time.append(t)
    jetson_temps.append(jetson.temp)
    heatsink_temps.append(heatsink.temp)

# Plot results
plt.figure(figsize=(10, 6))
plt.plot(time, jetson_temps, label='Jetson Orin', linewidth=2)
plt.plot(time, heatsink_temps, label='Heat Sink', linewidth=2)
plt.axhline(70, color='r', linestyle='--', label='Throttle Threshold')
plt.xlabel('Time (seconds)')
plt.ylabel('Temperature (Â°C)')
plt.title('Thermal Simulation: Jetson Orin with Heat Sink')
plt.legend()
plt.grid(True)
plt.savefig('thermal_sim.png')
plt.show()

print(f"Steady-state Jetson temp: {jetson.temp:.1f}Â°C")
print(f"Steady-state Heat Sink temp: {heatsink.temp:.1f}Â°C")
```

**Expected Output:**
```

Steady-state Heat Sink temp: 62.5Â°C
```

**Use Case:** Quick validation before ordering heat sink (cheaper than trial-and-error).
